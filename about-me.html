Article of the week: Semantic Elements

Why are semantic elements important? Semantic elements help make the website that you are building more usable and accessible. It helps your code communicate with the web browser. The definition of semantic (according to Webster) is “of or relating to meaning in language”. Semantic elements are HTML tags that make the code more readable to the developer. Because these tags are standard anyone can go into your code and understand it. Semantic elements focus more on meaning than style. For example, if you want a sentence bolded you would not use <b>, but instead <strong> since the developer wants the word to be more emphasized and strong correlates to meaning more than bold.
